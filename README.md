# BFR
 teensy controls code, team BFR, MAE 157A Spring 2023

### Usage
Soldered AltIMU10 to I2C 2 pins (24/25) on Teensy 4.1. Changed Wire definintions in libraries (`LPS`,`SparkFunLSM6DS3`,`LIS3MDL`) from `Wire` to `Wire2`.
